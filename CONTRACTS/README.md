## 📖 **راهنمای استفاده**

### **برای توسعه‌دهندگان:**
1. قبل از هر تغییر، `PREFLIGHT_CHECKLIST_CONTRACT.md` را مطالعه کنید
2. تغییرات را طبق `ATOMIC_EXECUTION_WORKFLOW_CONTRACT.md` اجرا کنید
3. از `DESIGN_PRINCIPLES_CONTRACT.md` برای UI/UX استفاده کنید
4. از `FormStandards.md` برای فرم‌ها استفاده کنید

### **برای هوش مصنوعی:**
1. همیشه `AI_COMPLIANCE_CONTRACT.md` را رعایت کنید
2. قبل از هر پیشنهاد، چک‌لیست پیش‌پرواز را اجرا کنید
3. تغییرات را به‌صورت اتمیک پیشنهاد دهید
4. منتظر تایید صریح کاربر باشید
